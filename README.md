# Flashcard App

A simple and intuitive flashcard application built with React and Next.js 14.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev

Open http://localhost:3000 with your browser to see the result.

You can start editing the page by modifying app/page.tsx. The page auto-updates as you edit the file.

Features

    Create and manage flashcards
    Organize flashcards into categories
    Study mode to test your knowledge
    Responsive design
    User authentication

Technologies Used

    React
    Next.js 14
    Node.js
    MongoDB
    Tailwind CSS
```
